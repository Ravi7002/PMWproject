# Lumina Blog Template

A premium, responsive blog post template built with React and Vite.

## Features

- **Modern Design**: Dark mode aesthetic with glassmorphism and vibrant gradients.
- **Responsive**: Fully responsive layout for all devices.
- **Components**:
  - Sticky Navbar with scroll effect
  - Rich Blog Post layout with typography optimization
  - Newsletter subscription section
  - Comprehensive Footer
- **Performance**: Optimized with Vite for fast HMR and build.

## Getting Started

1.  Navigate to the project directory:
    ```bash
    cd blog-app
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:5173](http://localhost:5173) in your browser.

## Customization

- **Styles**: Edit `src/index.css` to change the color palette and typography.
- **Content**: Update `src/components/BlogPost.jsx` with your own content.
- **Images**: Replace `public/hero.png` with your own hero image.
